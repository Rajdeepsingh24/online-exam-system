# 🧑‍💻 Online Examination Management System

## 📌 Project Description

This is a web-based Online Examination Management System developed using **Python (Django)** and **MySQL/SQLite**.
The system allows administrators to create and manage exams, while students can attempt exams online and view their results.

---

## 🚀 Features

### 👨‍🏫 Admin (Teacher)

* Create and manage exams
* Add, update, and delete questions
* View student attempts
* Monitor results

### 👨‍🎓 Student

* Register and login
* View available exams
* Attempt exams online
* View results instantly

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/online-exam-system.git
cd online-exam-system/backend
```

### 2. Create virtual environment

```
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3. Install dependencies

```
pip install django
```

### 4. Run migrations

```
python manage.py migrate
```

### 5. Run server

```
python manage.py runserver
```

### 6. Open in browser

```
http://127.0.0.1:8000/
```
